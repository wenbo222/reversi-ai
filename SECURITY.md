# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.x.x   | :white_check_mark: |

## Reporting a Vulnerability

Please do not open a public issue, pull request, or discussion around a security vulnerability. 
Instead, please report the vulnerability directly with the GitHub "private vulnerability reporting" 
feature here: <https://github.com/wenbo222/reversi-ai/security>.
